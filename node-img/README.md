## 步骤

+ 首先需要安装[GraphicsMagick](https://sourceforge.net/projects/graphicsmagick/files/latest/download?source=files)

+ 安装依赖模块

```
npm install
```

+ 启动运行

```
node app.js
```

+ 浏览器打开访问

```
http://127.0.0.1:3000/
```
